## 开发

1. 安装环境

> [node.js](https://nodejs.org/en/)
  
> [yarn](https://yarnpkg.com/zh-Hans/docs/install)

2. 安装依赖

``` javascript
$ yarn
```

3. 开发:

```
yarn run dev
```

4. 微前端模式开发(子应用需要启动本地服务):

```
yarn run qiankun
```

5. 测试

```
yarn run test
```
    
6. 发布

```
$ yarn run release 
```
